# 音樂榜
一起享受世界的聲音

# 音樂榜介紹
該作品採用 KKBOX 以及YouTube API 所製作，僅限前端開發練習 。

# GitHub Pages
https://alyciaciou.github.io/side-project-kkbox-demo/

# 簡介
*  綜合新歌歌單查看、試聽以及MV觀看
*  中、英、日、韓新歌歌單查看、試聽以及MV觀看
*  情境歌單查看、試聽以及MV觀看
*  歌曲查詢、試聽

#使用技術
*  KKBOX API
*  YouTube API
*  JavaScript
*  Bootstrap5
*  axios
*  scss
